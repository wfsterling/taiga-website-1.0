Taiga Market - Proof of Concept Demo

Taiga Market PoC is composed of two components:

  • Asset Builder - Allows the creation of a smart contract and its respective assets
  • Alien Invasion dApp Game - Will access the blockchain to load the spaceships the user owns in order to use in the game

Explained walkthrough of our demo

Asset Builder

In order to use these applications you will need to install Metamask or any other Web3 provider and connect to the "Ropsten Test Network". In case you don't have Ether, please request them via the Ropsten Faucet application.

The first step is to create the assets on the Asset Builder:

  1. Go to https://taigamarket.firebaseapp.com/
  2. Create your store

Once the store is created (corresponding to deploy a new smart contract on the Ethereum Network) we need to create each asset with the respective tokens.

We have prepared this demo with three assets: - TFigther - MFalcon - Enterprise

Each of these spaceships need to have different properties. You can customize your asset with the name, description and total supply you want. Please be aware that the price is not currently being used.

Here are three different JSON text for the properties field:

  • TFither "spriteImage": "images/sprites/tfighter.png"
  • MFalcon "spriteImage": "images/sprites/mfalcon.png"
  • Enterprise "spriteImage": "images/sprites/enterprise.png"

In case you want to create your own spaceship, you need to edit the following image and store in a server (or at IPFS): The JSON properties will need to be changed accordingly.

Once all the assets are created, they all belong to the owner.

We suggest you to create a second account on the Ropsten Network and use the bottom of the Asset Builder in order to transfer one of the tokens to the second account. The address of the smart contract needs to be copied from the store smart contract address which was populated when the store was created. You can inspect the console of your browser in order to know the TokenIds for each asset type.

Now that all the assets are created and transferred, you can login to the Alien Invasion dApp Game to play.

Alien Invasion dApp Game

In order to play this game with the assets you have just created, you need to do the following:

  1. Go to https://taiga-alieninvasion.firebaseapp.com
  2. Set your smart contract address created in the Asset Builder form
  3. Click reload in order to connect to the blockchain

The game will connect to the Ropsten Network and load the assets for the selected account on Metamask. If you change the account on the Metamask, please click the reload button again.

Have fun!
